# Dashboard-de-nalisis-predictivo-y-prescriptivo-del-abandono-de-clientes-en-servicios-de-telco
Dashboard integral en Power BI para el análisis de retención de clientes (Telecom Churn). Incluye flujo completo E2E: limpieza de datos (ETL), análisis exploratorio (EDA) para detección de patrones, y modelos de Machine Learning para analítica predictiva y prescriptiva.
